#  Option A Store

This is my **Project Phase One (Option A)** submission for the Web Development course.  
It is a static storefront built using **PHP** and **CSS**, with modular includes and a simple database structure.

---

##  Features
-  **Homepage**, **Shop**,  **About**,  **Contact**,  **Login/Register**
- PHP **includes** for `header.php` and `footer.php`
- Linked external **CSS** with color scheme and Google Fonts
- MySQL database structure file (`structure.sql`)
- Fully responsive layout

---

##  Technologies Used
- PHP  
- HTML5 & CSS3  
- Google Fonts (Playfair Display, Inter)  
- XAMPP for local hosting  
- Git & GitHub for version control
